# Simple React Calculator

A basic calculator web application built with React that can handle simple operations.

## Features
* Simple operations such as addition, subtraction, and multiplication.
* Clicking the equals button after an operation repeats the operation with the first number input being the previous result.
* Clicking another operation button after inputting the second number of an ongoing operation acts as an equals button and activates the new operation.
